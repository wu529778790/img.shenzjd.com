## 🎓 经验教训

### 成功经验

1. **设计先行** 💡
   - 先确定设计系统再编码
   - 使用 UI/UX Pro Max 技能获取专业建议
   - 统一的视觉语言提升品质

2. **组件复用** ♻️
   - 动画组件库提升效率
   - 一致的代码结构
   - 减少重复代码

3. **渐进优化** 📈
   - 逐步改进而非推倒重来
   - 保持功能稳定性
   - 持续改进体验

4. **性能关注** ⚡
   - 从一开始就考虑性能
   - GPU 加速动画
   - 图片懒加载

5. **无障碍优先** ♿
   - 从设计阶段就考虑 A11y
   - 语义化 HTML
   - 键盘导航支持

### 踩坑记录

1. **HTML 结构错误** 🐛
   - `<p>` 不能包含 `<div>`
   - 注意标签嵌套规则
   - 使用验证工具

2. **Base UI 组件层级** 📚
   - 必须查阅组件文档
   - 正确的嵌套结构
   - Context 依赖

3. **Framer Motion 嵌套** 🔄
   - 避免嵌套 button 元素
   - 使用 motion.div
   - 保持语义化
