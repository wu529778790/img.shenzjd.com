# 设置页分组优化设计

## 背景

当前设置页是单栏流式布局，4 个设置组堆叠在一列里：外观主题 → 图床配置 → 危险操作 → 关于。随着功能增加（图片统计、操作日志等），这种扁平流式布局会越来越长，查找和切换成本高。

## 目标

重构为**系统设置风格**：左侧分类导航 + 右侧内容区，让用户能在视觉上快速定位和切换设置组。移动端使用底部/顶部 Tab 导航。

## 方案

### 布局方案

**桌面端（≥768px）：** 左侧固定侧边栏 + 右侧自适应内容
```
┌─────────────────────────────────────────────────────────┐
│  ⚙️ 设置                                            [主题] │
├──────────────┬──────────────────────────────────────────┤
│  外观        │  主题模式                                │
│  图片处理    │  ☀️ 浅色  🌙 深色  💻 跟随系统            │
│  网络        │                                          │
│  危险操作    │  默认压缩质量        [━━━●━━━━━] 80%    │
│  账户        │  自动压缩            [●] 开启             │
│  关于        │  ...                                     │
└──────────────┴──────────────────────────────────────────┘
```

**移动端（<768px）：** 顶部水平滚动 Tab
```
┌────────────────────────────────────────────────────┐
│  ⚙️ 设置                                     [主题] │
├──────────────┬───────────────────────────────────────┤
│ [外观][图片][网络][危险][账户][关于]  ← 水平滚动   │
├──────────────┴───────────────────────────────────────┤
│  主题模式                                             │
│  ☀️ 浅色  🌙 深色  💻 跟随系统                        │
└──────────────────────────────────────────────────────┘
```

### 分类结构

| 分类 | 图标 | 包含内容 | 边框色 |
|------|------|---------|--------|
| 外观 | Sun | 主题模式（浅色/深色/跟随系统） | 默认灰 |
| 图片处理 | Image | 压缩设置（质量 + 自动开关） + 水印设置 | 默认灰 |
| 网络 | Globe | CDN 服务 + 使用 Raw 链接 | 默认灰 |
| 危险操作 | Shield / AlertTriangle | 清空配置 + 退出登录 | 红色 |
| 账户 | User | （占位，后续扩展） | 默认灰 |
| 关于 | Info | 版本 + 描述 + 技术栈 | 默认灰 |

### 技术实现

- **激活状态管理**：`useState<number>`（当前激活分类索引），零 Zustand 改动
- **响应式**：`< md` 用顶部滚动 Tab，`>= md` 用左侧固定导航
- **动画**：`framer-motion` `AnimatePresence` 做切换内容区淡入/滑入动画
- **侧边栏导航项**：点击切换 active 索引，active 项有 primary 色背景 + 左侧指示条
- **底部移动端 Tab**：与侧边栏共享同一状态，响应式渲染不同结构

### 组件拆分

```
SettingsPage (page.tsx)            — 布局容器 + 激活状态
└── SettingsNav                    — 侧边栏导航项列表（桌面）/ 顶部 Tab（移动）
└── SettingsContent                — 右侧内容区容器（含切换动画）
    ├── AppearanceSection          — 外观主题
    ├── ImageProcessingSection     — 压缩 + 水印
    ├── NetworkSection             — CDN + Raw
    ├── DangerSection              — 危险操作
    ├── AccountSection             — 账户（占位）
    └── AboutSection               — 关于
```

### 状态管理

```tsx
// SettingsPage
const [activeSection, setActiveSection] = useState(0)

// 分类配置
const sections = [
  { id: 'appearance', label: '外观', icon: Sun },
  { id: 'image',      label: '图片处理', icon: Image },
  { id: 'network',    label: '网络', icon: Globe },
  { id: 'danger',     label: '危险操作', icon: ShieldAlert },
  { id: 'account',    label: '账户', icon: User },
  { id: 'about',      label: '关于', icon: Info },
]
```

### 变更文件

| 文件 | 改动 |
|------|------|
| `src/app/settings/page.tsx` | 重构为侧边栏布局（拆分 sections 数组 + 响应式结构 + 切换动画） |

### 不涉及

- 不改 `configStore`、`useConfigStore`
- 不改任何 API 层
- 不修改已有功能逻辑，仅做布局和结构重组
- AccountSection 仅做占位 UI，不新增功能
