## 🌟 动画效果总览

### 动画时长配置
```typescript
fast:   150ms  // 按钮点击
normal: 250ms  // 标准过渡
slow:   400ms  // 复杂动画
stagger: 30-50ms // 列表交错
```

### 动画类型

| 类型 | 使用场景 | 示例 |
